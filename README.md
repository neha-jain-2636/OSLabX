# OSLabX

An interactive Operating System simulator that demonstrates core concepts of operating system.

## Modules

- **System Calls**
- **File Allocation**
- **File Organization**
- **Disk Scheduling**
- **Process Syncronization**
- **Deadlocks**
- **CPU Scheduling**
- **Memory Management**
- **Contiguous Memory Allocation**
- **Page Replacement**

Extra Modules not in Lab Syllabus:
  
- **Real Time Operating System**
- **Memory Management Unit**
