Why we built this

Concepts you can see.

OS coursework is dense with diagrams that never move. We rebuilt those diagrams as living simulators so you can poke at them, break them, and learn how they actually behave.

01

Step-by-step execution

Pause, advance and replay every algorithm to follow exactly how state evolves.

02

Honest metrics

Average waiting time, seek distance, page faults — surfaced as you iterate.

03

Edge cases included

Belady's anomaly, DAG cycles, fragmentation — the failure modes are first-class.

04

Zero install

Pure HTML, CSS and JavaScript. Open any module and it just runs.
